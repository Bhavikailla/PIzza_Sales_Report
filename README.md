# PIzza_Sales_Report built using SQL + Excel + Power BI + Tableau + Python
* Pizza Sales Analysis is a data analytics project that analyzes pizza store sales transactions to understand revenue, customer behavior, and sales trends.

* The project calculates key KPIs such as Total Revenue, Total Orders, Total Pizzas Sold, Average Order Value, and Average Pizzas per Order.

* Sales performance is analyzed based on pizza category, size, and type to identify the most popular products.

* Time-based analysis (daily, hourly, and monthly trends) is performed to understand peak sales periods and seasonal demand.

* The project identifies top 5 best-selling pizzas and bottom 5 least-selling pizzas to help optimize the menu and marketing strategies.

* Interactive data visualizations and dashboards are created using Python in Jupyter Notebook to provide insights and support business decision-making.
